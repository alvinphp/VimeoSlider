<h3>Installation and use</h3>
<ul>
<li>Install the plugin using Joomla! Extension Manager.</li>
</ul>

<ul>
<li>Look for the extensions manager and go to the module.</li>
</ul>

<ul>
<li>Configure module.</li>
</ul>
